# Cálculo de Entropía

Este repositorio contiene el desarrollo de algoritmos para el análisis de fuentes de información, cálculo de entropía y métodos de codificación eficiente (Huffman).

## Contents
* **Calculo de entropia:** Solución integral de los ejercicios de cálculo de entropía para diferentes fuentes.
* **Algoritmo de Huffman:** Implementación para la generación de códigos de longitud variable.
* **Lectura de Imagen y Audio:** Módulos para extraer datos de archivos multimedia y analizar su contenido informacional.

## Tecnologías Utilizadas
* **Lenguaje:** C# (.NET 8.0 / Framework 4.7.2)
* **IDE:** Visual Studio 2022
* **Librerías:** Microsoft Cognitive Services (para procesamiento de audio).

## Instrucciones de Uso
1. Clonar el repositorio.
2. Abrir el archivo `.sln` en Visual Studio.
3. Restaurar los paquetes NuGet (especialmente los de Azure Speech Services).
4. Ejecutar el proyecto de consola para ver los resultados de los cálculos en la terminal.
